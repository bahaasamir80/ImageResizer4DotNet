This is a .NET library written in C# to resize images at three
different qualities of low, medium, and high. The advantage of
this library is that the high quality mode produces images
identical to the output of professional software like Paint.NET.

The implementation is described in details on my blog:

http://keyvan.io/very-high-quality-image-resizing-in-net
